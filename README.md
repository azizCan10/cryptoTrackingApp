## Crypto Tracking Spring Boot App

**Technologies**

* Java - Spring Boot
* Lombok

You can see a crypto price with its symbol

`localhost:8080/v1/crypto/avgPrice/MATICUSDT`

```json
{
    "mins": 5,
    "price": 0.85199977,
    "symbol": "MATICUSDT"
}
```
