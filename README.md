# Crypto Tracking Spring Boot App

## Technologies

* Java - Spring Boot
* Spring Data JPA
* Lombok
* PostgreSQL
* Binance API

## Features

* Sign up
* Add any crypto in Binance to your watchlist
* See any crypto price in Binance
